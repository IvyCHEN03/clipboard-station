# 灵感悬浮球虚拟讲解版脚本

目标时长约 98 秒。镜头采用虚拟讲解者加真实产品界面的双摄结构。讲解者保持静态自然表情，只做轻微镜头呼吸和左右机位切换，不使用循环假口型。

## 1. 开场

**画面：** 浅蓝色灵感球出现，讲解者从右侧入镜。

**旁白：** 我最近总在好几个 AI 之间来回跑。灵感很多，剪贴板只有一个，多少有点不讲武德。

## 2. 自动接住复制内容

**画面：** 连续复制文字、截图和表格；相同文字连续出现三条。

**旁白：** 所以我做了灵感悬浮球。文字、截图、表格，按下 Command C，它就顺手接住；同一句复制三次，也会老老实实留下三条。

## 3. 搜索和筛选

**画面：** 输入搜索词，依次选中两个标签，再切换今天、3 天和鱼的 7 天记忆。

**旁白：** 东西多了也不怕。搜一句话，点几个标签，再选今天、三天或者鱼的七天记忆，想找的那条很快就浮上来。

## 4. 当前结果批量操作

**画面：** 在筛选结果内全选、复制和倒带，隐藏内容保持不动。

**旁白：** 筛完以后，全选、复制、删除和倒带都只管眼前这些，不会隔着筛选条件误伤后面的内容。

## 5. 积木组合

**画面：** 把两个文字片段拖进空白组合框，在积木之间出现光标并输入过渡句。

**旁白：** 看到顺眼的句子，就拖进组合框。积木中间可以直接补字，先写一句过渡，再塞下一块，顺序完全按你的脑回路来。

## 6. AI Polish

**画面：** 点击 Polish，积木内容变成一段连贯全文，原积木仍保留。

**旁白：** 懒得自己捋逻辑，就点 Polish。它只拿当前组合的内容去润色，补上衔接，原来的积木和顺序都还在。

## 7. 图片复制和拖拽

**画面：** 灵感球图片区先保持空白。按下 Command C 后出现一条图片栏；抓住缩略图拖进右侧文档，落点显示图片本身。

**旁白：** 图片这里我重新做了一遍。Command C 复制一张图，它会直接变成灵感球里的一栏，不是文件路径。抓住这张图片往文档里一拖，放进去的就是图片本身。

## 8. 多图帖子收图

**画面：** 打开多图帖子，Command 点击悬浮球，当前帖子图片叠成一行暂存。

**旁白：** 碰到小红书或者公众号这种多图帖子，也不用右键二十次。Command 点一下悬浮球，当前帖子自己的图片会一起叠进暂存栏。

## 9. 单张选择和 PNG 保存

**画面：** 双击暂存行，图片展开；取消一张，保留三张并保存。

**旁白：** 双击这一行摊开，哪张心动就勾哪张，再保存成真正的 PNG。头像、表情包和重复图，会尽量替你挡在外面。

## 10. 保存完整网页

**画面：** 点击新按钮“存网页”，先显示处理状态，再出现 `页面.html` 和 `完整页面.png` 两个文件。

**旁白：** 这次还在更新一个很实用的小按钮：存网页。点一下，同时留下可搜索的 HTML 和从头到底的完整网页截图。

## 11. 七天记忆和回忆浅滩

**画面：** 展示今天、3 天、快到 7 天的进度条；删除内容进入回忆浅滩并可恢复。随后展示固定位置、自定义快捷键和开机启动。

**旁白：** 鱼的七天记忆会提醒你什么时候快忘了；删错的内容先去回忆浅滩，不会立刻消失。固定位置、快捷键和开机启动，也都能按自己的习惯调。

## 12. 收尾

**画面：** 回到浅蓝色灵感球和 GitHub 地址。

**旁白：** 内容默认留在本地，AI 也是你点了才用。它不替你想，只负责把那些差点跑掉的灵感，先稳稳接住。
