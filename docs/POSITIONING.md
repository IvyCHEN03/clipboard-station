# Product Positioning

Linggan Floating Ball is not trying to be a general clipboard history replacement. It is a focused station for AI-heavy work where the user needs to collect, compare, reorder, and compose many fragments.

## Best Fit

Use it when you:

- Work across several AI chats at the same time.
- Copy repeated fragments because order and context matter.
- Need screenshots, table-like text, and plain text in one visible place.
- Want to assemble a final prompt or answer from smaller blocks.
- Prefer local-first storage over cloud clipboard sync.

## Not The Main Goal

It is not optimized for:

- Replacing every clipboard manager.
- Syncing clipboard history across devices.
- Managing long-term notes or knowledge bases.
- Automating every macOS workflow.
- Running AI tagging without explicit provider configuration.

## Compared With Other Tool Types

| Tool type | Usually optimized for | Linggan Floating Ball focuses on |
| --- | --- | --- |
| Clipboard history managers | Recall the last copied items | Collect AI-work snippets and compose a final prompt |
| Launchers such as Raycast or Alfred | Commands, search, and broad automation | A small persistent station for copied AI fragments |
| Cloud clipboard tools | Sync across devices | Local-first privacy and explicit optional AI tagging |
| Notes apps | Long-term writing and organization | Temporary working memory while comparing AI outputs |
| Screenshot managers | Image capture and annotation | Screenshot snippets alongside text and tables |

## Product Bet

The bet is that AI-heavy work creates a new clipboard problem: the useful unit is no longer just the last copied item, but a small set of fragments that need to stay visible, tagged, reorderable, and paste-ready.

That is why the app emphasizes:

- The floating bubble instead of fragile global shortcuts.
- Repeated capture instead of aggressive deduplication.
- Colored composition blocks instead of a plain clipboard list.
- Local encrypted persistence instead of cloud sync.
- Optional AI tags instead of mandatory network behavior.
