# Code of Conduct

Linggan Floating Ball is a small local-first tool for people working with private clipboard material, AI prompts, screenshots, and notes. The community should feel careful, practical, and safe.

## Expected Behavior

- Be respectful and specific.
- Assume people are trying to help.
- Keep feedback focused on the product, code, docs, tests, or user workflow.
- Use redacted examples when discussing clipboard content, screenshots, API providers, or logs.
- Welcome first-time contributors and small scoped improvements.

## Unacceptable Behavior

- Harassment, insults, threats, or personal attacks.
- Posting someone else's private clipboard content, screenshots, credentials, local files, or API keys.
- Pressuring users to share sensitive content to reproduce a bug.
- Repeated off-topic comments after maintainers ask to refocus.
- Spam, self-promotion, or unrelated support requests.

## Privacy-Sensitive Reports

Do not paste private clipboard contents, API keys, encrypted local state, or screenshots with sensitive information into public issues or pull requests.

For privacy-sensitive bugs, describe the behavior with dummy content. Follow [SECURITY.md](SECURITY.md) for security and private-data reports.

## Enforcement

Maintainers may edit, hide, or remove comments that expose sensitive data or violate this code. Maintainers may also close issues, lock threads, or block accounts for repeated or severe violations.

If you see a problem, open an issue with a minimal description or contact a maintainer through the repository owner profile. Do not include private payloads in the report.

## Scope

This code applies to GitHub issues, pull requests, discussions, documentation, release feedback, and any project-maintained community space.
